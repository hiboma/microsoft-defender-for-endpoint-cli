# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it through GitHub's **Private Vulnerability Reporting** feature:

1. Go to the [Security Advisories](https://github.com/hiboma/mde-cli/security/advisories) page
2. Click **"Report a vulnerability"**
3. Fill in the details and submit

Please do **not** open a public issue for security vulnerabilities.

## Response

We will acknowledge receipt within 7 days and provide an update within 30 days.
